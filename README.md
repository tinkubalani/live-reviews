# live-reviews

Live review application

# Pre-requirements for running projects locally:

Node.js must be installed
NPM must be installed

# Frontend

-> To install the project dependencies, navigate to the project root directory (live-reviews/frontend) and run the following command:
npm install

# FE Usage:

To run the project locally, use the following command:
npm run start:dev  
This command will start the frontend server on port 3000.
Frontend code will run on the following server URL: http://localhost:3000/

or

cd live-reviews/frontend
npm i
npm run start:dev

# Backend

-> To install the project dependencies, navigate to the project root directory (live-reviews/backend) and run the following command:
npm install

# Usage:

To run the project locally, use the following command:
npm run start:dev  
This command will start the backend server on port 5000.
Backend APIs will run on the following server URL: http://localhost:5000/

or

cd live-reviews/backend
npm i
npm run start:dev
